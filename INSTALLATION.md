INSTALLATION
============

Prerequisites
-------------

- Rust toolchain (rustup). Install from https://rustup.rs/

Getting started
---------------

1. Clone the repository:

   git clone git@github.com:Hitomatito/Orbit.git

2. Enter the project directory:

   cd Orbit

3. Build the project:

   cargo build --release

4. Run tests:

   cargo test

Notes
-----

- If you need a specific Rust version, create a `rust-toolchain` file.
- For development, use `cargo run` to execute the binary (if present).
